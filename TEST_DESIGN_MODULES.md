# 功能模块测试方案

## 1. 测试概述

本测试方案针对重复文件分析工具的核心功能模块进行全面测试，确保每个模块的功能正确性和稳定性。

### 1.1 测试目标
- 验证核心功能模块的功能正确性
- 验证模块间的交互正确性
- 验证边界情况的处理
- 验证错误处理的完整性

### 1.2 测试范围
- FileScanner - 文件扫描模块
- HashCalculator - 哈希计算模块
- IndexManager - 索引管理模块
- ConfigManager - 配置管理模块

## 2. FileScanner 模块测试

### 2.1 测试用例：扫描单个文件
- **测试目标**：验证单个文件扫描功能
- **测试步骤**：
  1. 创建测试文件
  2. 调用 `scan_file()` 方法扫描文件
  3. 验证返回的文件信息正确
  4. 验证文件信息包含：文件名、扩展名、大小、创建时间、修改时间、访问时间
- **预期结果**：扫描成功，文件信息完整准确

### 2.2 测试用例：路径排除功能
- **测试目标**：验证路径排除功能
- **测试步骤**：
  1. 创建测试文件和目录
  2. 配置排除模式
  3. 调用 `is_path_excluded()` 方法
  4. 验证排除逻辑正确
- **预期结果**：排除功能正确，匹配的路径被排除

### 2.3 测试用例：扫描目录
- **测试目标**：验证目录扫描功能
- **测试步骤**：
  1. 创建测试目录和多个文件
  2. 调用 `scan_directory()` 方法扫描目录
  3. 验证扫描结果包含所有文件
  4. 验证扫描统计信息正确
- **预期结果**：扫描成功，所有文件都被扫描

### 2.4 测试用例：从CSV导入
- **测试目标**：验证从CSV导入文件列表功能
- **测试步骤**：
  1. 创建测试CSV文件
  2. 调用 `scan_from_csv()` 方法导入

  4. 验证导入统计信息正确

### 2.5 测试用例：大文件处理

  4. 验证文件名正确处理
- **预期结果**：扫描成功，特殊字符正确处理

### 2.7 测试用例：权限错误处理
- **测试目标**：验证权限错误处理
- **测试步骤**：
  1. 创建无权限访问的文件
